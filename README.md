### CRUD

- Пакет [json-server](https://github.com/typicode/json-server)
  - npm i --save-dev json-server - установка в терминале
  - "server": "json-server --watch db.json" - прописать в папке package.json ->
    в разделе scripts
  - npm run server - запуск в терминале
